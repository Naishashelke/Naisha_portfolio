Projects

AI Chatbot for E-Commerce Customer Support

Developed a scalable AI chatbot system designed to automate customer support for e-commerce platforms.

The system was implemented using Angular for the front-end interface, Spring Boot for backend services, and MySQL for data management. Robotic Process Automation (RPA) tools were integrated to streamline backend operations and improve efficiency in handling customer queries.

The chatbot was designed to enhance customer engagement by providing automated responses to frequently asked questions and support requests.

Raspberry Pi Smart Water Prediction System

Designed an Internet of Things (IoT) based system to predict underground water availability in highland regions using Raspberry Pi and sensor modules.

The system collects environmental and soil parameters through sensors and processes the data using a Python-based predictive model to estimate potential underground water presence.

This project demonstrated the integration of IoT hardware systems with machine learning techniques for environmental monitoring applications.

Object Detection using Convolutional Neural Networks

Developed a computer vision system using Convolutional Neural Networks (CNN) to perform object classification and detection tasks.

The project involved training deep learning models to recognize and categorize objects from image datasets and evaluating performance using precision, recall, and F1-score metrics.

This work helped strengthen understanding of deep learning architectures and real-time computer vision applications.

Travel Buddy Automation System

Developed an automation-based travel assistance system using UiPath StudioX for workflow automation.

The system automated travel booking processes and integrated APIs for retrieving travel information. It also implemented a notification system to provide real-time updates on bookings and travel schedules.

This project demonstrated the application of robotic process automation (RPA) in simplifying repetitive tasks in digital platforms.
